# 🚀 Fundamentos de JavaScript

***Este repositório reúne exemplos práticos e didáticos dos principais conceitos fundamentais da linguagem JavaScript, com foco em lógica de programação e construção de base sólida para desenvolvimento.***

## 📚 Conteúdos Abordados
***🔹 Condicionais***

***Estruturas responsáveis pela tomada de decisão no código.***

- if
- else if
- else
- switch
- Declaração com let, const e var
- Operadores:
- Aritméticos (+, -, *, /)
- Comparação (==, ===, >, <)
- Lógicos (&&, ||, !)


Este projeto tem como objetivo:

- Fortalecer a lógica de programação
- Praticar conceitos essenciais do JavaScript
- Servir como material de apoio para iniciantes
- Compor portfólio profissional no GitHub

## 🛠️ Tecnologias Utilizadas
- JavaScript (ES6+)
- Node.js (opcional para execução)
